Politique de confidentialité

Dernière mise à jour : juin 2026

1. Introduction

La présente politique de confidentialité décrit comment l'application Word collecte, utilise et protège vos données personnelles.

En utilisant Word, vous acceptez les pratiques décrites dans ce document. Si vous n'acceptez pas ces conditions, veuillez cesser d'utiliser l'application.

2. Données collectées

Données de compte, si vous créez un compte : adresse e-mail, nom d'affichage, photo de profil Google ou Facebook. Ces données sont utilisées pour identifier votre compte et personnaliser votre expérience.

Données de jeu : statistiques de parties, scores, progression, parties sauvegardées. Ces données sont stockées localement sur votre appareil et, si vous êtes connecté, dans Firebase Firestore pour la synchronisation.

Préférences et paramètres : thème, langue, options de jeu. Ces données sont stockées localement sur votre appareil.

Données d'utilisation anonymisées : Firebase Analytics peut collecter des données anonymisées sur l'utilisation de l'application afin d'améliorer les fonctionnalités. Ces données ne permettent pas de vous identifier personnellement.

3. Données non collectées

Word ne collecte pas vos informations bancaires ou de paiement, vos données de localisation géographique, vos contacts, ni vos données biométriques.

Les achats intégrés sont traités par Google Play. Word n'a pas accès à vos informations de paiement.

4. Utilisation des données

Vos données sont utilisées pour fournir les fonctionnalités de l'application, synchroniser votre progression entre appareils si vous êtes connecté, améliorer l'application grâce à des données anonymisées, et permettre les fonctionnalités sociales comme les classements ou défis.

Vos données ne sont jamais vendues à des tiers.

5. Services tiers

Firebase, Google LLC : authentification, base de données, analytique. Politique de confidentialité de Google : https://policies.google.com/privacy.

Google Play Billing : traitement des achats intégrés, régi par les conditions d'utilisation de Google Play.

Facebook Login, Meta : connexion optionnelle via Facebook. Politique de confidentialité de Meta : https://www.facebook.com/policy.php.

6. Conservation des données

Les données locales sont conservées sur votre appareil jusqu'à la désinstallation de l'application.

Les données de compte Firebase sont conservées tant que votre compte est actif. Vous pouvez demander la suppression de votre compte depuis les paramètres de l'application.

7. Vos droits

Conformément au RGPD, si applicable, vous disposez de droits d'accès, de rectification, d'effacement, de portabilité et d'opposition au traitement.

Pour exercer ces droits, contactez-nous à : serge.simono27@gmail.com.

8. Sécurité

Nous mettons en œuvre des mesures techniques appropriées pour protéger vos données : communications chiffrées, stockage sécurisé Firebase, et absence de stockage d'informations bancaires dans Word.

9. Mineurs

Word n'est pas destiné aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles provenant d'enfants de moins de 13 ans.

10. Modifications

Cette politique peut être mise à jour. La version en vigueur est celle disponible dans l'application et sur cette page web.
